# نظام استخراج درجات الطلاب 📊

نظام ويب متطور يسمح للطلاب باستخراج درجاتهم من خلال رقم الطالب والرقم السري، مع قاعدة بيانات **backend** قوية.

## المميزات ✨

- ✅ **Backend قوي**: خادم Node.js + Express لتخزين آمن للبيانات
- ✅ **قاعدة بيانات JSON**: تخزين البيانات في ملفات JSON على الخادم
- ✅ **تسجيل دخول الطلاب**: دخول آمن برقم الطالب والرقم السري
- ✅ **عرض الدرجات**: عرض درجات الطالب في جميع المواد
- ✅ **لوحة التحكم**: إضافة ورفع ملفات Excel بسهولة
- ✅ **دعم المواد المتعددة**: دعم أعداد غير محدودة من المواد
- ✅ **تصدير البيانات**: تحميل نسخة احتياطية بصيغة Excel
- ✅ **واجهة عربية**: واجهة مستخدم تدعم اللغة العربية

## المتطلبات 📦

- Node.js (نسخة 12 أو أحدث)
- npm (يأتي مع Node.js)

## التثبيت والتشغيل 🚀

### 1. تثبيت المتطلبات

```bash
npm install
```

هذا سيثبت المكتبات التالية:
- **express**: إطار العمل الرئيسي
- **cors**: للسماح بطلبات من مصادر أخرى
- **multer**: لرفع الملفات
- **xlsx**: لقراءة ملفات Excel

### 2. تشغيل السيرفر

```bash
npm start
```

أو للتطوير مع التحديث التلقائي:

```bash
npm run dev
```

### 3. فتح التطبيق

افتح المتصفح وادخل إلى:
```
http://localhost:5000
```

## إعداد ملف Excel 📋

يجب أن يكون ملف Excel بالصيغة التالية:

| الرقم | الاسم | الرقم السري | الرياضيات | اللغة الإنجليزية | اللغة العربية |
|------|-------|----------|----------|-----------------|-------------|
| 101   | أحمد محمد | 1234 | 95 | 88 | 92 |
| 102   | فاطمة علي | 5678 | 87 | 90 | 85 |
| 103   | محمود حسن | 9012 | 78 | 82 | 80 |

**ملاحظات مهمة:**
- الأعمدة المطلوبة: **الرقم**، **الاسم**، **الرقم السري**
- يمكن استخدام أسماء بديلة: (ID, Student ID) و (Name) و (Password, كلمة المرور)
- باقي الأعمدة تُعتبر مواد دراسية
- يجب أن تكون الدرجات أرقام

## كيفية الاستخدام 💡

### للطلاب:
1. اختر "دخول الطالب"
2. أدخل رقم الطالب والرقم السري
3. اعرض درجاتك

### للمسؤول:
1. اختر "دخول المسؤول"
2. أدخل كلمة المرور: `admin123` (غيّر هذا في `server.js` للأمان)
3. اختر ملف Excel ورفعه
4. عرض جميع البيانات
5. تحميل نسخة احتياطية (backup)
6. حذف جميع البيانات (إذا لزم الأمر)

## API Endpoints 🔌

- `GET /api/getStudents` - الحصول على جميع الطلاب
- `GET /api/students/:id` - الحصول على بيانات طالب معين
- `POST /api/upload` - رفع ملف Excel/CSV
- `POST /api/students` - إضافة/تحديث طالب واحد
- `DELETE /api/deleteAll` - حذف جميع البيانات
- `DELETE /api/students/:id` - حذف طالب معين
- `GET /api/export` - تصدير البيانات كـ Excel

## تخزين البيانات 💾

البيانات يتم حفظها في:
```

```

هذا الملف يُنشأتلقائياً عند تشغيل السيرفر، ويحتفظ بجميع بيانات الطلاب والدرجات.

## الأمان ⚠️

**تنبيهات مهمة:**
1. غيّر `ADMIN_PASSWORD` في `server.js` إلى كلمة مرور قوية
2. استخدم HTTPS عند النشر على الإنترنت
3. لا تشارك كلمات المرور مع أحد
4. استخدم متغيرات البيئة (env variables) لحماية البيانات الحساسة

## استكشاف الأخطاء 🐛

### الخادم لا يعمل:
```bash
# تأكد من أن المنفذ 5000 غير مشغول
# أو غيِّر المنفذ في server.js

```

### خطأ في تحميل البيانات:

## نصائح مهمة ⚠️

1. **نسخة احتياطية**: قم بتحميل نسخة احتياطية دوريًا من البيانات
2. **أمان المتصفح**: لا تشارك رابط التطبيق مع أشخاص غير معنيين
3. **تنظيف البيانات**: تذكر أن البيانات تُحفظ محليًا في المتصفح فقط
4. **اختبار الملف**: جرب ملف Excel صغير أولاً للتأكد من تنسيقه

## حل المشاكل 🔧

### المشكلة: الطالب لا يستطيع تسجيل الدخول
**الحل:** 
- تأكد من أن رقم الطالب والرقم السري مطابقان تمامًا (حروف صغيرة/كبيرة)
- تأكد من أن ملف Excel تم رفعه بنجاح

### المشكلة: الدرجات لا تظهر
**الحل:**
- تأكد من أن الدرجات مكتوبة كأرقام في Excel
- تأكد من عدم وجود نصوص إضافية مع الأرقام

### المشكلة: الأعمدة الإضافية لا تظهر
**الحل:**
- تأكد من أن أسماء الأعمدة صحيحة
- دقق في عدم وجود مسافات زائدة
